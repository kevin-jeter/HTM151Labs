User Guide

Welcome to the HTM151: Nuvolo for Managers Labs! This guide is designed to help you navigate through the various labs included in this course. Each lab focuses on a specific aspect of using Nuvolo, providing hands-on experience and practical knowledge.

Getting Started

  1. Access Nuvolo Development Environment: Use the DEV environment for all labs. Login via https://vaoittest.servicenowservices.com with your VA.gov account.
  2. Navigate This Guide: Use the dropdown menu above to select a specific lab. All sections include step-by-step instructions, screenshots, and interactive elements like input fields for answers.
  3. Theme Toggle: Switch between light and dark mode using the toggle in the top right for better readability.

Tips for Success

  • Explore Freely: Don't hesitate to click around Nuvolo—use the "History" tab to backtrack if needed. Avoid using your browser's back button, as it may not work well with Nuvolo.
  • Document Your Work: Many labs include input fields for recording answers (e.g., work order numbers). Use these to track your progress and share with instructors.
  • Troubleshooting: If you encounter issues, you're not alone! Ask your instructor(s) for help and you'll be helping out your peers too!
  • Practice Hypotheticals: Labs encourage creating hypothetical work orders and assets to build confidence without real-world consequences.

If you have questions, reach out to your instructor or refer to the course materials. Happy learning and mastering Nuvolo!

Happy learning!

Lab 1: Technician Schedules and Rates

In this lab, you will learn how to navigate the Nuvolo system and understand technician schedules and rates.

Part 1: User Attributes

  1. In the "All" section, search for "User Attributes" and select "Nuvolo User Attributes" under the Clinical Asset Management Admin Module.
  2. Search and click on your Name via filtering the Name column
  3. Select your Name and review the following data fields for accuracy:
    • Facility: The facility you report to on an organizational chart
    • Duty Station: The facility you are performing work actions at currently
    • HTM User. Make sure this box is checked to ensure you have access to Nuvolo
    • HTM Manager. Make sure this box is checked to ensure you have manager privileges in Nuvolo
    • HTM Primary Role. Your user role, as applicable
  4. Once confirmed, select "Update" (only necessary if you make changes)
  5. Repeat this steps for at least 2 additional technicians at your site

Part 2: Technician Schedules

  1. In the "All" filter navigator, search for "Technician Schedules" and select it.
  2. We're going to create a New Technician Schedule. Select the "New" button.
  3. Fill out the following fields on the form using your OWN name.
    • Name: Your Full Name - HTM151 Lab
    • Technician: Your Full Name (Use the magnifying glass to search)
    • Schedule Type: Shift
    • Description: HTM151 Technician Schedule Example
  4. Let's find your schedule! In the "Name" search bar, enter "*" and then your Last name
  5. Select your Technician Schedule from the list and click on "Load Related Lists" at the bottom of the page.
  6. Let's create a new Schedule Entry. Click on "New" in the Schedule Entries related list.
  7. Fill out the following fields on the Schedule Entry form:
    • Name: Your Name Monday-Friday
    • Type: --None--
    • Show as: Busy
    • When: Today's Date and Your Work Hours
    • Repeats: Every Weekday
    • Repeat Until: Leave Blank
    • Type: Leave Blank
  8. Submit the Schedule Entry. You'll see the schedule entry appear in the related list.

Part 3: Rates

Let’s make sure you know how to accurately view and update your technician’s pay rates within Nuvolo. This ensures proper record keeping to see how much labor cost is being put into specific devices. It will help make decisions on device maintenance and upkeep versus device replacement.

  1. In the "All" filter navigator, search for "Internal Employee Rates" and select it.
  2. Let's create a new Employee Rate. Select the New button.
  3. We're going to create a new employee rate. Fill out the following fields on the Employee Rate form:
    • Employee: Kevin Jeter
    • Position Entry Date: Today's Date
    • Facility: Your Facility
  4. Submit the Employee Rate.
  5. Search and find the employee rate you just created using the Employee search (Hint: type *Jeter)
  6. Open the entry by selecting the blue hyperlink under "Yearly."
  7. We need to calculate Kevin's Labor rate. To do so, follow these steps:
    • Annual Salary: $40,000
    • Divide this by 2080: $40,000 / 2080 = $19.23 (round to the nearest cent)
    • Enter $19.23 into the "Labor Rate" field
  8. Finally, let's fill out the rest of Kevin' Internal Rate Information. Fill out the following fields:
    • Pay Plan: GS
    • Classification Series: Enter the job series. (0858 is for BME)
    • Grade: Enter Pay Grade
    • Step: Enter current pay grade step
    • Role: Enter HTM Role
  9. Select "Update" and you've completed updating the employee rate.
  10. Repeat these steps for at least two additional employees.

Lab 2: Clinical Device Verification and Work Order Assignment

Part 1: Navigation and Primary Data Fields

  1. In the "All" filter navigator, search for "Unverified Devices" and select it. You may need to add "Verified" as a column to see the unverified devices.
  2. Click on the first Unverified Device to view its details. Hint: You must select the Asset Tag to open the device!
  3. Review the primary data fields on the device details page. Based on the information, answer the following questions:
    1. What is the device's serial number?
    2. Is the device HTM Supported or Non-HTM Supported?
    3. What is the Warranty Expiration Date?
    4. What is the PM Support Group?
    5. What's the total Cost of Ownership?
    6. When was the last activity on this device?
    7. Is this device Cerner Compatible?
  4. Before verifying this device, let's find out how to automatically assign groups for PM and CM Support on this device. Select the "Resources" tab.
  5. Let's update the PM Support Group. Select the "Eyeglass" icon next to the PM Support Group field.
  6. In the pop-up window, select one of the groups to give them automatic assignment to this device.
  7. Finally, let's verify this device. Select the "Verified" button and select Save.

Part 2: Work Order Assignment

We just used Automatic Assignment in the last lab, but how do we manage our groups and create new ones if we need to?

  1. In the "All" filter navigator, search for "Groups" and select it (Hint: It's the one under "Self Service")
  2. In the list, search for "HTM" followed by the HTM Program TLA (in this example, we use ABQ. Yours will be different!) and select Enter
  3. Select one of the groups to update.
  4. At the bottom of the page, you can select "Load Related Lists" to view additional details like Roles, Group Members, Groups, Skills, etc.
  5. Review the data fields and answer the following questions:
    1. What is the purpose/description of this group?
    2. How many Group Members are in this group?
    3. How many different Roles are assigned to this group?
    4. Is the group Active?
    5. What is the Group email?
    6. Does it have a group parent? If so, what is it?
  6. Challenge: Add a member to the group! (Hint: Look for the "Edit" button in a related list)
